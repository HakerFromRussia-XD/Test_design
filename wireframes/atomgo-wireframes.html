AtomGo · Wireframes

Средне-точные вайрфреймы (414×896, iOS) для двух пайплайнов: клиент и админ. Дизайн-система AtomGo задаёт цвета статусов, форму карточек и тени; вайрфреймы серые — раскраска и иконография придут на этапе hi-fi.
Экранов: 9 (1 клиент + 8 админ) Сетка: 414×896 iPhone 11 Pro Max Подсказки: синие пины с пояснениями
Клиентский пайплайн

Один экран — аренда + оплата

1 экран
01Моя аренда
Велосипед, оплачено до, ставка → красная плашка долга → шторка снизу с тарифами «1 день / 1 неделя / 2 недели / 1 месяц / весь долг»
Админский пайплайн

Дашборд аренд → детали → клиенты → велосипеды

8 экранов
02Все аренды (дашборд)
Сводка сверху + поиск + чипы статусов + список аренд. Цвет рамки иконки = статус (зелёный/жёлтый/фиолетовый).
03Смена статуса
Поп-овер тапом по иконке: «Долгосрочная / Вернут в течении недели / Велосипед у меня».
04Детали аренды
Велосипед, арендатор → карточка клиента, журнал CHARGE/PAYMENT/ADJUSTMENT, корректировка и завершение.
05Все клиенты
Поиск по ФИО / телефону / паспорту, фильтры «Должники» и «Активные», группировка по букве.
06Карточка клиента
Контакты + документы (видео, договор, комментарий) + список аренд этого клиента.
07Новый клиент
Обязательные: ФИО, телефон, паспорт, адрес, логин/пароль (генерация). Опционально: видео, договор, коммент.
08Все велосипеды
Сводка + группировка «Свободные / В аренде». Карточка показывает фото, модель, SN, ставку.
09Новый велосипед
Обязательные: модель, VIN, мотор, ставка W. Опционально: серийники АКБ 1 и АКБ 2.

Допущения и заметки

  • Один холст, 9 фреймов. Каждый ряд можно прокручивать горизонтально. Tweaks-панель (тогл в тулбаре) меняет плотность и переключает подсказки.
  • Tab-bar 4 секции: Дом (аренды) · Велосипеды · Клиенты · Настройки.
  • Поиск клиентов: один инпут ищет по ФИО, телефону и паспорту одновременно. Фильтры «Должники / Активные» — чипы под поиском.
  • Статус аренды — 3 цветные рамки вокруг фото велика; поп-овер для смены — экран 03.
  • Журнал — линейный список с тегом типа (CHARGE/PAYMENT/ADJUSTMENT), знаковой суммой и датой.
